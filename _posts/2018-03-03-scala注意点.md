1.区分大小写  

2.类名的第一个字母要大写，驼峰式

```
class myFirstScalaClass
```
3.方法名，所有方法名称的第一个字母用小写，驼峰式

```
def myMethodName()
```

4.程序文件名-程序文件的名称要与对象名称完全匹配

```
if object name is: HelloWorld
保存的文件名应是:HelloWorld.scala
```

5.scala程序从main()方法开始处理
```
def main(args:Array[string])
```
